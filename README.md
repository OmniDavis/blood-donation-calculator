Una aplicación web sencilla que te permite conocer con qué tipos de sangre eres compatible para donar y recibir, según tu grupo sanguíneo.

- Descripción
Esta calculadora muestra de forma visual e inmediata:

A quién puedes donar sangre según tu tipo
De quién puedes recibir sangre según tu tipo

Cubre los 8 tipos sanguíneos del sistema ABO con factor Rh: O−, O+, A−, A+, B−, B+, AB− y AB+.

- Cómo usarlo

Abre la aplicación en tu navegador
Selecciona tu tipo sanguíneo en el menú desplegable
Haz clic en "Calcular compatibilidad"
Los resultados aparecerán indicando a quién puedes donar y de quién puedes recibir

Disclaimer

Esta herramienta es únicamente con fines educativos e informativos. 
La compatibilidad mostrada está basada en el sistema ABO con factor Rh,
pero la compatibilidad real para transfusiones sanguíneas depende de múltiples factores adicionales, 
como pruebas cruzadas, anticuerpos irregulares y otras consideraciones clínicas.

**No utilices esta calculadora para tomar decisiones médicas.** 

Cualquier procedimiento relacionado con donación o transfusión de sangre debe ser evaluado y confirmado por personal médico calificado.
